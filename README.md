### Renewal 이유
종종 이용하는 항공사이지만, 브랜드만의 개성이 없고 메인화면에서 제공하는 정보가 부족하여 Renewal을 결심하였다.

### Renewal Point
Point 1. 이스타항공 로고 색상을 메인 컬러로 선택.

Point 2. 사용자가 서브 메뉴를 일일이 누르지 않더라도, 메인 화면에서 원하는 정보를 쉽게 찾을 수 있도록 크게 시각화함.

Point 3. 각 section 별로 다양한 움직임을 적용하여 사용자가 지루하지 않도록 함.

### ISSUE

처음으로 하는 팀 프로젝트.

시작하기 전에는, 한 사이트를 여러 명이 나눠서 담당하기 때문에 금방 완성할 수 있을 거로 생각했다. 그러나 이것은 크나큰 착각 !!!

-  레이아웃 짜기
 : 각자 생각하는 항공사의 이미지가 달라서 레이아웃 구성이 어려움.

- 차이 나는 작업 속도
 : 개인의 성격 또는 일의 우선순위에 따라 작업 속도가 달랐으며, 이 문제는 완성 속도를 늦추었다.

 - 암호 같은 클래스 명
 : 개인 작업하던 습관대로, 본인이 편한 단순 알파벳으로 class와 id를 작성.


이 모든 일의 원인은 하나였다.

소통의 부재!
<br/>

### 해결 방법
#### - 팀 프로젝트는 소통이 생명!

- 여러 항공사 사이트를 방문 후, 부족 또는 추가하고 싶은 부분을 고려하여 의논 후 레이아웃 작성.

- 작업 시작 사흘 후, 차이 나는 작업 속도 확인. 내가 다른 팀원의 main page 담당 부분과 sub page 한 개 더 작업하기로 협의.

- 최종 점검 시 다른 팀원의 파일을 봤을 때, 단순 알파벳으로 된 class와 의미 없는 id 확인. 또 다른 팀원과 둘이 이 팀원의 모든 파일 전면 수정함.
